# Security Policy

## Reporting a Vulnerability

DBView runs entirely in the browser and stores all data locally in `localStorage`. No data is ever sent to any server.

If you discover a security vulnerability, please report it privately by opening a draft security advisory on GitHub:

https://github.com/Vanshbordia/dbview/security/advisories/new

Please do not disclose security vulnerabilities publicly until they have been addressed.

## What to Include

- A clear description of the issue
- Steps to reproduce (if applicable)
- Any potential impact

## Response Time

You can expect an acknowledgment within 48 hours, and a fix timeline will be communicated shortly after triage.
